숫자야구 게임
=============
- **목표** : 컴퓨터가 생성한 3자리 숫자를 플레이어가 맞추는 게임
- **숫자에 대한 요구사항**
    - 각 자리는 1-9 사이의 숫자
    - 동일한 숫자는 사용될 수 없다. 즉, 숫자는 중복되지 않아야 한다.
    - ex) 333, 112, 119 `불가능`
- **게임 결과**
    - 각 시도에 대해 '스트라이크'와 '볼'의 개수를 출력하여 플레이어가 숫자를 맞출 수 있도록 유도한다.


lv1 게임 기본 기능 구현
-----------------------
- 정답 숫자 생성
- 사용자 입력과 정답을 비교해 strike, ball 갯수 출력
- 정답을 맞추면 축하메시지와 함께 게임 종료
